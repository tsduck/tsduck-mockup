==== target_smartcard_descriptor

Defined by DVB in <<ETSI-301-192>> and <<ETSI-102-006>>.
Must be in a UNT (table id 0x4B) or INT (table id 0x4C).

[source,xml]
----
<target_smartcard_descriptor super_CA_system_id="uint32, required">
  <!-- Private data bytes -->
  Hexadecimal content
</target_smartcard_descriptor>
----
